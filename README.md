这是一个前端基于Vue3框架后端基于Express框架并使用sqlite3作为数据库的个人游戏网页, 目前只有一个基于canvas实现的横版游戏，玩家可以读取其他玩家的评分评论，也可以自行对游戏进行评分评论，后台可对评论进行修改删除
